Data Analysis & Learning Methods - Lecture 3

▶ What kind of feature selection methods are there?
Filter methods, wrapper methods, embedded methods.
▶ What's feature selection?
Feature selection is the process of selecting a subset of features from the original set of features.
▶ What's an example of feature selection?
Selecting the most relevant features for a given task.
▶ What's feature extraction?
Constructing new features from existing features.
▶ What's an example of feature extraction?
Instead of using millions of raw pixels to recognize an image, we compute features such as: Edges, Color histograms, Texture features, etc...
